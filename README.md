# Imagehelpers
Rudimentary image manipulation (((framework)))

Some things may or may not work correctly.

Some example usage in tests.

## Features
- Easy manipulation of individual pixel components
- Image loading/saving
    - [x] PNG
    - [ ] JPG
    - [ ] BMP
    - [ ] FF
- Resizing
    - [x] Nearest neighbour
    - [x] Bilinear
    - [x] Trilinear
    - [x] Bicubic
    - [ ] Lanczos
    - [ ] Catmull-Rom
    - [ ] Cubic Hermite
- Rotating
    - [ ] Vertically, Horizontally
    - [ ] Radial
- Drawing
    - [x] Line
    - [x] Circle
    - [ ] Ellipse
    - [ ] Bezier curve
- Filtering
    - [x] Greyscale
    - [x] Negative
    - [x] Sepia
    - [x] Quantization
    - Convolutional kernel routine
        - [x] Smoothing
        - [x] Sharpening
        - [x] Edge detection
        - [x] Blur
- [ ] Dithering
    - [x] Some kernels
- [ ] Documentation
